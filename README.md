Neofetch inspired, CLI-tool for view your system information.

### Support:
- [x] Linux
- [ ] MacOS
- [ ] Windows

##

## Installation:
### Clone using Git
```bash
git clone https://github.com/reilinux/xifetch
cd xifetch
python src/main.py
```
### Copy the binary to /usr/bin
```
sudo cp bin/xifetch /usr/bin
```
